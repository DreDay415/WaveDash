# Component Inventory

## Layout Components
1. **DashboardShell** - Main container with sidebar and header
2. **Sidebar** - Collapsible navigation (icon-only mini mode)
3. **Header** - Top bar with breadcrumbs, search, notifications
4. **MainContent** - Content area with max-width constraint

## Metric Display Components
5. **KPICard** - Large stat card with icon, value, trend, sparkline
6. **SystemHealthGauge** - Circular gauge (0-100 score)
7. **DiskUsageCard** - Progress bar with projection
8. **SyncStatusIndicator** - Live pulsing status badge
9. **CountdownTimer** - Next sync countdown with animation

## Data Visualization Components
10. **SyncTimelineChart** - Last 24 hours activity (Recharts LineChart)
11. **SuccessRateGauge** - Semi-circular gauge
12. **CallVolumeHeatmap** - 24-hour heatmap grid
13. **TopCallersTable** - Sorted leaderboard with avatars
14. **DailyTrendChart** - 7-day bar chart

## Interactive Components
15. **TimeRangeToggle** - 24H / 7D / 30D switcher
16. **RefreshButton** - Manual refresh with spinner
17. **ThemeToggle** - Dark/Light mode (start with dark)
18. **SearchCommand** - ⌘K command palette

## Utility Components
19. **LoadingSkeleton** - Shimmer loading states
20. **EmptyState** - "No data" placeholder
21. **AlertBanner** - Critical warnings (disk space < 20%)
22. **Tooltip** - Hover info (Radix Tooltip)

## Animation Components
23. **PulseIndicator** - Animated dot for live status
24. **CountUp** - Animated number transitions
25. **ProgressRing** - Animated circular progress
